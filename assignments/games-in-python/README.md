
# 📘 Assignment: Games in Python - Hangman

## 🎯 Objective

Build a Hangman game in Python to practice strings, loops, conditionals, and user input handling.

## 📝 Tasks

### 🛠️ Build the Core Hangman Game

#### Description
Create the main game flow where a word is selected, the player guesses one letter at a time, and the game reveals progress after each guess.

#### Requirements
Completed program should:

- Select a word randomly from a predefined list.
- Show the hidden word progress using underscores (example: _ _ _ _).
- Accept one-letter guesses from the player and update the revealed letters.


### 🛠️ Track Attempts and Finish Conditions

#### Description
Implement the rules for incorrect guesses, remaining attempts, and game-over outcomes.

#### Requirements
Completed program should:

- Track and display remaining attempts after incorrect guesses.
- End the game when the player guesses the full word or runs out of attempts.
- Display clear win and loss messages to the player.
